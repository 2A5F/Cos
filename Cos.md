## 变量

### 定义

```js
var a = 1;
var b: int;
```

### 修改

```js
let a = 2;
let a + 1;
```

## 函数

### 定义

```rust
fn foo() {}
fn add(a: int, b: int) -> int { a + b }
```

### 调用

```js
foo();
add(1, 2);
```

## 基础类型

### 对象

```js
var o = {
  a = 1,
  b = 2,
};
```

### 数组

```js
var a: [int] = [1, 2, 3];
```

### 元祖

```js
var t: (int, bool, str) = (1, true, 'asd');
```
